# Modul Pembelajaran SPK (AHP, SAW, WP, TOPSIS)
Modul pembelajaran yang menerapkan Descicion Support System yang mudah digunakan. Dikembangkan dengan menggunakan bahasa PHP yang populer sehingga mudah dalam melakukan setup. Pengguna dapat dengan cepat membandingkan beberapa metode DSS valid seperti AHP, SAW, WP serta TOPSIS.

Aplikasi web sederhana ini berbasis PHP Native untuk mempelajari alur perhitungan Sistem Pendukung Keputusan.
Modul ini mencakup metode:
- AHP (Analytic Hierarchy Process) - Lengkap dengan perhitungan Matrix & CR
- SAW (Simple Additive Weighting)
- WP (Weighted Product)
- TOPSIS
